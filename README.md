# webScore React プロジェクト

Vite + React で構築したシンプルなスターターです。`npm install` で依存関係を取得し、`npm run dev` で開発サーバーを起動できます。

## セットアップ

```bash
npm install
npm run dev
```

ブラウザで `http://localhost:5173` を開くとアプリを確認できます。

## ビルド

本番ビルドは次のコマンドで生成できます。

```bash
npm run build
```

ビルド内容は `dist/` に出力されます。`npm run preview` を使うとローカルでビルド結果を確認できます。
